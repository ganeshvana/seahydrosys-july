Schedule Activity to Multiple Users
====================================
* The "Schedule Activity to Multiple Users" feature in Odoo 15 allows you to assign an activity or task to multiple users within the system. This feature is particularly useful when you have a task that requires collaboration or involvement from multiple individuals or teams.

Installation
============
- www.odoo.com/documentation/15.0/setup/install.html
- Install our custom addon
